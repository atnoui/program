Day 1

July 22

الأساسيات: هيكل HTML The Foundation: HTML Structure

نبدأ بالهيكل العظمي للويب. ستتعلم كيف تنظّم المحتوى وتستخدم الوسوم بشكل صحيح. We start with the skeleton of the web — structure content logically and set up your dev environment.

  • إعداد VS Code والإضافات الأساسيةSetting up VS Code and essential extensions
  • فهم DOM وأساسيات وسوم HTMLUnderstanding the DOM and basic HTML tags
  • HTML الدلالي لإمكانية الوصول وـSEOSemantic HTML for accessibility and SEO
  • النماذج والمدخلات وجمع بيانات المستخدمForms, inputs, and capturing user data
  • بناء الهيكل الخام للمشروع النهائيBuilding the raw structure of your final project

Day 2

July 23

التصميم والتفاعل: CSS وجافاسكريبت Style & Spark: CSS + JavaScript

أضف الحياة لهيكلك — التصميم بـCSS والتفاعل بـJavaScript. Bring your structure to life with CSS styling and JavaScript interactivity.

  • نموذج الصندوق في CSS، Flexbox، والتصميم المتجاوبCSS Box Model, Flexbox, and responsive design
  • تنسيق العناصر بخصائص CSS الحديثةStyling elements with modern CSS properties

  • تحديد عناصر DOM والتحكم فيهاSelecting and manipulating DOM elements
